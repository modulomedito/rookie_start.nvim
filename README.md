# rookie_start.nvim

Single file configuration, just clone and run, then get neovim ready

## Installation

### Linux and macOS

```bash
git clone https://github.com/modulomedito/rookie_start.nvim.git "${XDG_CONFIG_HOME:-$HOME/.config}"/nvim
```

### Windows

Using PowerShell

```powershell
git clone https://github.com/modulomedito/rookie_start.nvim.git "${env:LOCALAPPDATA}\nvim"
```

Using cmd.exe

```cmd
git clone https://github.com/modulomedito/rookie_start.nvim.git "%localappdata%\nvim"
```
